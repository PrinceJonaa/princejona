## 2024-05-23 - Focus Styles Standardization
**Learning:** Interactive elements require explicit `focus-visible` rings using `ring-foreground` and `ring-offset-background` to handle both light/dark modes and keyboard accessibility effectively.
**Action:** Apply `focus:outline-none focus-visible:ring-2 focus-visible:ring-foreground focus-visible:ring-offset-2 focus-visible:ring-offset-background` to all interactive elements (Links, Buttons).
